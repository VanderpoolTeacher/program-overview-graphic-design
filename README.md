# DC Graphic Design Program Overview

Interactive single-page presentation for the Defiance College Graphic Design major. Built as a standalone HTML/CSS/JS file with no framework dependencies.

## Views

- **Overview** — Program stats, domain descriptions, and key design highlights
- **Program MLOs** — 8 Bloom-aligned Major Learning Outcomes across 4 domains (Conceptual, Technical, Applied, Professional), with drill-down to see which courses address each MLO
- **Courses** — 13 core courses with expandable CLO details and MLO mapping, filterable by level
- **Alignment Matrix** — Full course-to-MLO alignment grid with hover highlighting
- **Plan of Study** — Four-year, 120-credit degree plan (Catalog 2025-26) with semester-by-semester course layout

## Features

- Dark and light mode toggle
- Font Awesome 6 domain icons
- Color-coded domains (Conceptual, Technical, Applied, Professional)
- Responsive layout for mobile and desktop
- Accessible typography (12pt minimum, 1.5 line spacing)

## Usage

Open `index.html` or `program-overview.html` directly in a browser. No build step or server required.
